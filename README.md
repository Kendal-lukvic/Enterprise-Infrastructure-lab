# 🛡️ Enterprise-Infrastructure-lab Lab

## Overview

This project documents my hands-on Active Directory home lab built to strengthen my Windows Server administration and IT skills.

The goal of this lab is to simulate a real enterprise Windows environment and practice the tasks commonly performed by IT Support Specialists, System Administrators, and Cybersecurity Professionals.

---

## Technologies

- Windows Server 2022
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy
- PowerShell
- VirtualBox

---

## Learning Objectives

By completing this project, I will learn how to:

- Deploy Windows Server
- Install Active Directory
- Configure DNS
- Configure DHCP
- Create Users and Groups
- Create Organizational Units (OUs)
- Apply Group Policy Objects (GPOs)
- Join Windows clients to a domain
- Automate administration using PowerShell
